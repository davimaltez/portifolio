# Portifólio Pessoal | Davi Maltez
Este repositório contém o código-fonte do meu portfólio pessoal, utilizando HTML5 e CSS puro, focado em demonstrar minhas habilidades desenvolvidas ao longo dos meus estudos e experiências, organizar e facilitar a vizualização dos meus projetos e compartilhar minha trajetória profissional.

## Design

optei por um design mais escuro com detalhes em vermelho para remeter a uma vibe "Netlfix", para que passasse a sensação de que você tivesse vendo uma série da minha vida
